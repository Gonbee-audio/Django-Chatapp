# Django-Chatapp

起動するには

```
 $ docker-compose build
```

```
 $ docker-compose up
```

で動かせます。



使っている技術 html.css.pyhton(django),javascrpit(vue.js),aws ec2,docker

チャットアプリです。

簡単な機能紹介。
・チャットの送受信
・いいね機能
・コメント機能
・プロフィール編集機能
・個別チャット機能
